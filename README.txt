* MedievalClicker *

¿Cómo funciona?
- El juego és un clicker-rpg inspirado en pokéclicker. Está dividido en tres secciones: una UI (arriba izquierda), 
un visual de las zonas (arriba derecha) y un mapa para desplazarse (abajo). 
- Seleccionamos cada zona a la que queremos ir y en las zonas donde hay monstruos si clicamos la pantalla inflingiremos daño
- Por cada enemigo derrotado recibes botín: oro y pociones. Usa el oro en la tienda (mapa iniciar y de el medio) para aumentar 
el daño inflingido y la vida total. 
- Arma y escudo son distintos segun "upgrade". 
- Autoclicker: vale 500 de oro.
